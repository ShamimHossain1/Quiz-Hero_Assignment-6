Question and Answer

1) Difference between map, filter, foreach, find.

Ans: map creates a new array from calling a function for every array element.
filter returns an array containing the element that satisfies the condition.
forEach provide callback function once for each element in an array in ascending-index order.
find returns the element itself that satisfies the condition.

2) Difference between local storage, session storage, and cookies.

Ans: Local storage is not session based,need to be deleted via JS or manually.
Session storage is session based and working per window or tab.
Cookies expiry depends on the setting and working per window or tab.

3) What is event loop and how it's work?

Ans: JavaScript has a runtime model based on an event loop, which is responsible for executing the code, collecting and processing events, and executing queued sub-tasks.
